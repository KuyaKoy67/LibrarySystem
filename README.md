Library System, Project Description
Author: Santos Carl Nathan

Library Management System, which is used to maintain the online record of books, DVDs, and magazines along with their user including students, teachers, administrator in Java. It is built on an object-oriented architecture with inheritance and polymorphism to impose certain borrowing rules and limits for the various users. This system is a hybrid of dual search means: that is, modern Java Streams and old school recursion for quickly finding items within a collection. Any data is persisted through CSV files using a strong loadData() that brings the application in sync with current state at startup and backupData() that saves the actual catalog and user activity when exiting
